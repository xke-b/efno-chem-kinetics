# C2H4: soft OH weighting is safer, but not a real solution

_Date: 2026-04-24_

I tried the softer follow-up after the nearby radical-weighting failures:
- keep the early-window corrective data
- keep intermediate weighting
- add only a **small** `OH` weight
- leave `HO2` alone

## What happened

This was better behaved than the stronger radical-weighted branches.

It did **not**:
- blow up `HO2`
- or cause the catastrophic global instability of the strong `OH`-only branch

That is good.

But it also did not become the new best branch.

## What I think the result means

The soft-`OH` branch looks like a **safer compromise**, not a real fix.

That matters because it suggests the current problem is no longer likely to yield to one more nearby weight tweak. The easy weighting gains are probably mostly exhausted.

So the next step should likely be something more structured than another small manual profile change.
