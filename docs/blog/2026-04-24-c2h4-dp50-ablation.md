# Tighter pressure filtering is not automatically better for C2H4

_Date: 2026-04-24_

I ran the next obvious target-refinement ablation for C2H4: tighten the case-pair pressure filter from `|Δp| <= 100 Pa` to `|Δp| <= 50 Pa` and see if the learned run gets closer to stock.

The result is informative and a little counterintuitive:
- the tighter filter still runs cleanly to `5e-6`
- it still beats the fully unfiltered case-pair model on some broad metrics
- but it does **not** beat the `|Δp| <= 100 Pa` model overall

In particular, the tighter `50 Pa` model has:
- higher mean `Qdot` than the `100 Pa` model
- worse temperature drift than the `100 Pa` model
- a broader pressure tail than the `100 Pa` model

So the response is non-monotonic. A stricter pressure filter is not automatically a better chemistry target.

That is useful because it rules out one simple continuation path. The next refinement should probably change the subset logic itself—most likely a mixed filter such as `|Δp| <= 100 Pa` plus a modest `|ΔT|` cap—rather than just squeezing pressure harder.
