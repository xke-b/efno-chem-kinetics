# efno-chem-kinetics

Agent-assisted reproduction and extension of EFNO-style methods for stiff chemical kinetics, with emphasis on generalization, autoregressive stability, and CFD-coupled usefulness.

## Repository focus
- PDF-to-structured-notes workflow for technical papers
- faithful EFNO reproduction from the paper
- benchmark-first evaluation
- CFD-coupled testing and improvement studies
- public documentation of findings through a docs site

## Documentation
The project documentation is published with GitHub Pages from the `docs/` directory and includes experiment notes, findings, and technical blog posts.

- Website: https://xke-b.github.io/efno-chem-kinetics/
